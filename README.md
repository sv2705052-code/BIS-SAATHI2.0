# BIS2.0 — Demo static site

This repository contains a minimal static site ready for deployment to Vercel.

Deployment (manual):

1. Install the Vercel CLI (optional):

```bash
npm i -g vercel
```

2. From this project folder run:

```bash
vercel login
vercel --prod
```

Or connect this Git repository in the Vercel dashboard and deploy from there.

Files of interest:

- `index.html` — site entry
- `styles.css` — simple styling
- `vercel.json` — Vercel configuration
